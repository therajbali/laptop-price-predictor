## Project Overview
This project analyzes Airbnb listing data from a German city to identify 
patterns related to availability, room types, neighborhoods, and host behavior.

The goal is to generate data-driven insights that can help Airbnb hosts 
better understand market competition and make informed listing management decisions
within a regulated environment.

---

## Business Context
Airbnb hosts in Germany operate under strict local regulations and strong competition.
Understanding how listings are distributed and managed across the city can support:

- Better availability planning
- Strategic choice of room types
- Awareness of local competition
- Differentiation between casual and professional hosts

---

## Dataset
- Source: Airbnb listings dataset (provided externally)
- Rows: ~17,700 listings
- Key features:
  - Neighbourhood
  - Room type
  - Availability (days per year)
  - Reviews and review activity
  - Host listing count
  - License information

### Data Limitations
- `price` column contains 100% missing values and was excluded
- `neighbourhood_group` contains 100% missing values and was excluded
- Analysis focuses on descriptive insights only
